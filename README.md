# Ankur Varshney — Senior Enterprise Analytics Consultant Portfolio (V3)

Welcome to the **Portfolio V3** repository of **Ankur Varshney**, Senior Data Analytics Consultant & BI Architect with 11+ years of experience delivering enterprise BI, cloud data pipelines, and AI-enabled analytics.

## Overview
This repository contains the V3 Single Page Application (SPA) built with Vanilla HTML5, CSS3, and JavaScript, featuring:
- **Senior Enterprise Consultant Positioning**: Enterprise scale indicators, industry badges (Insurance, Banking, Retail, Aviation, Power, Financial Services), and business outcome metrics.
- **Surfaced Technical Insights**: 5 flagship engineering write-ups near the top before the full Knowledge Hub.
- **3-Tier Skill Matrix**: Core Expertise, Strong Working Knowledge, and Familiar Technologies.
- **Business-First Case Studies**: 10 enterprise projects detailing problems, challenges, solution architecture, and quantifiable impacts.
- **Embedded Interactive Tableau Showcase**: Live visualization widgets integrated directly from Tableau Public.
- **AI Resume Assistant & Command Palette**: Interactive chatbot assistant (`Ctrl+K` palette navigation).

## Local Development & Viewing
Simply open `index.html` in any web browser.
